# DropDownMenuView 下拉菜单

标签选择器下拉菜单,通过代理设置数据, 支持高度自适应,支持双链表联动tableView和collectionView,链表宽度设置,通过DMConfiguration配置样式,使用方便简单

#Requirements

```
* Swift 5
```

```
* iOS 10.0 +
```
